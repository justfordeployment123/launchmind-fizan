💡 TROUBLESHOOTING QUICK REFERENCE:

Problem: "API key not found"
→ Check .env file exists in launchmind/ directory
→ Check variable names match exactly (case-sensitive)
→ Run: echo $ANTHROPIC_API_KEY (verify it's set)

Problem: "Bad credentials" (GitHub)
→ Check GITHUB_TOKEN is correct (copy exactly)
→ Token may have expired - regenerate a new one
→ Verify token has 'repo' scope

Problem: "not_in_channel" (Slack)
→ Bot hasn't been invited to #launches
→ Type in #launches: /invite @LaunchMindBot

Problem: "Invalid email" (SendGrid)
→ SENDGRID_FROM_EMAIL must be verified in SendGrid
→ Go to Settings → Sender Authentication → verify your email

Problem: "Channel not found" (Slack)
→ #launches channel doesn't exist
→ Create it: Channels → + → Create

Full troubleshooting: See README.md → Troubleshooting section
